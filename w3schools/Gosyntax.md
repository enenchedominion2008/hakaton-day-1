       _______________ GOLANG SYANTAX _____________

in golang we have the package main 

we also have the import packages and also we have function main then we have the function body 

eg 
    package main
import ("fmt")

func main() {
  fmt.Println("Hello World!")
}


package main == This is my program (the starting point).
import "fmt" == I need this to print words on the screen.
func main() == This is where the magic happens. Go will start running from here.
fmt.Println("Hello World!") == This line prints "Hello World!" on the screen.
