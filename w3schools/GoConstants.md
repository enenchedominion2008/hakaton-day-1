Go Constants - Simple Summary
A constant is a value that cannot be changed after it is declared.

Use the const keyword to create a constant.
The value must be assigned at the time of declaration.
Once set, you cannot change it.

Syntax:
Goconst CONSTNAME = value
Example:
Goconst PI = 3.14
Important Points:

Constant names usually use UPPERCASE letters (e.g. PI, MAX_VALUE)
Follow the same naming rules as variables
Can be declared inside or outside a function
There are two types: Typed and Untyped constants

Typed Constant Example:
Goconst A int = 1
Key Rule: Constants are fixed forever — they are read-only.
